title: 2008년 6월 17일날 firefox 3 정식버젼이 릴리즈됩니다.

 http://www.mozillazine.org/talkback.html?article=23871 에 따르면 2008년 6월 17일에 드디어 firefox 3 가 정식으로 릴리즈됩니다.
 분명 Download day 이벤트에 참여했었는데 이상하게도 아직 메일이 오지 않았네요. 혹시나 싶어서 공식 홈페이지에 들어갔더니 떡하니 저런 포스팅이 11일에 벌써 되어 있었습니다.
 firefox 3 에서 주목할 만한 기능은 웹초보님의 포스팅을 참고하시고, 제일 기대되는 것은 역시 빨라진 자바 스크립트로 인해 구글 웹 오피스가 빨라질 거라는 것과 메모리를 적게 쓰고, 북마크 기능의 강화가 될 듯 합니다.
